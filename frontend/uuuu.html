Bienvenue sur notre application de gestion de projets

Sur cette page, vous pourrez consulter les informations sur les projets et les utilisateurs, ainsi que gérer les projets et les tâches associées.

Projets

Vous pouvez consulter les détails sur les projets en cliquant sur l'onglet Projets dans le menu de navigation. Vous pourrez également créer de nouveaux projets en cliquant sur le bouton Créer un nouveau projet.

Utilisateurs

Vous pouvez consulter les détails sur les utilisateurs en cliquant sur l'onglet Utilisateurs dans le menu de navigation. Vous pourrez également vous enregistrer ou vous connecter en cliquant sur les boutons correspondants.
